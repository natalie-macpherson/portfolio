Project Description

Refresh Website

The purpose of this project was to creation of a company and then creating a SAAS website to market and sell the companies service, which allows the user to record their calorie budget to maintain a healthy lifestyle.

Design Solution

This project incorporated UI/UX and web development elements in order to design and code a full website with all of the steps a user would have to follow to learn and sign up for the service.

Icons to illustrate features

The colours chosen for the website were colours which reflected a positive and active lifestyle. The imagery also reinforces this element and promotes the idea of this is where you want to be and this is what you need to do to get there. All the icons where created to visually represent all of the features provided in the program and provide a pop of colour but maintains its core message of living an active lifestyle.
